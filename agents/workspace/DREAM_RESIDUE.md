_(This file is populated by the nightly dream consolidation. It holds the 3-5 sharpest insights that emerged from last night's processing. Each insight has a 24-hour lifespan — the dream cron clears and rewrites this file each night.)_

### Format (written by dream cron each night):

```
### 1. [emoji] [Title — Short Description]
[1-3 sentence insight. Concrete. What it means for today's behavior.]

### 2. ⚠️ [Warning Title]
[What the pattern is, what the risk is, what action to take.]

### 3. 💡 [Insight Title]
[The reframe or connection that changes understanding.]
```

**Signal types:**
- 🌱 **Seed** — early pattern, worth watching
- 🔗 **Connection** — unexpected link between separate things
- ⚠️ **Warning** — risk if this pattern continues
- 💡 **Insight** — breakthrough understanding
